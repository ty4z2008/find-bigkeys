# find-bigkeys
redis tools. find big keys in redis

Usage:
```bash

   ___         ___       __            __
  / _ \___ ___/ (_)__   / /____  ___  / /__
 / , _/ -_) _  / (_-<  / __/ _ \/ _ \/ (_-<
/_/|_|\__/\_,_/_/___/  \__/\___/\___/_/___/ 0.0.1


Usage:
  -db int
        redis db (default 0)
  -p string
        redis password (default is empty)
  -url string
        redis server url (default "localhost:6379")
```
